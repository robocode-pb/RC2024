Шаблон цього проекту створений для студентів курсу Web Junior задля виконання завдань по Maker Time.

Проект містить базову структуру папок та файлів, яка використовувалась у навчальних матеріалах протягом року.

У файлі index.html вже є підключенн до стилів (style.css), локального скрипа (script.js).
А також, вже підключені jQuery та FontAwesome.

Усю необхідну інформацію по створенню веб-сторінок та веб-додатків можна знайти у роздаткових матеріалах.

Номери тем, які можуть бути корисними:

№10. DOM. Зміна документа. створення елементів
    Заняття, на якому розбирають JavaScript функції створення та додавання елементів на стронку, що може бути корисним під час виконання технічного завдання.

№21. jQuery. Маніпуляції DOM (To do list)
    Заняття, на якому розбирають jQuery функції створення та додавання елементів на стронку, що може бути корисним під час виконання технічного завдання.

№29. jQuery Project. Онлайн-квест 
    Заняття, на якому розбирають принцип роботи localStorage. Це сховище даних, яке дозволяє зберігати дані навіть після закриття браузера. Це може бути актуально для проекта 3-го рівня.
